# Androidstudio工具
工欲善其事必先利其器
## 1. 编译界面高亮设置

- 首先打开这个网站
     - 点击这里：[ColorThemes](http://color-themes.com/)
     - 进入网站后，可以看到很多样式
     - 接着，就可以选择喜欢的主题了
     - 凭自己高兴了，喜欢哪个下载即可
- 导入方式
     - 下载主题：xxx.jar
     -  注意：
      如果我们下载下来的jar名字如果有空格，一定要把空格去掉，同时文件路径中不要含有中文
     - 选择File—>Import Settings—>把刚才下载好的jar包导入
     - 重启Android Studio 在导入包后，会弹出2个界面，选确定或者OK即可


## 2. 修改字体大小

- 1、 File->settings
- 2、 Editor->Color Scheme->Color Scheme Font
   -  [ ] Use color scheme font instead the default 点击前面的方框
- 3、在Size后面更改字体大小，根据自己喜欢，选择字体的大小 (其他不用改)
- 4、 改好后，先点击Apply，然后点击OK

>这样就设置好了
